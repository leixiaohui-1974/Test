## 测试概述

本次测试验证了明渠边坡稳定性系统在各种阶跃变化场景下的响应特性。

### 测试方法

1. **初值计算**: 使用恒定流求解器计算初始水力条件
2. **非恒定流模拟**: 采用Preissmann隐式格式求解圣维南方程
3. **耦合模拟**: 水动力学、水质、边坡稳定性三模块耦合
4. **动态可视化**: 生成时间序列图和沿程分布GIF动画
